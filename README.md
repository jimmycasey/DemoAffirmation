# Live Demo Affirmation
A bash script that performs a pre-live-demo "affirmation" prayer with **ASCII art** of computing legends (Turing, Ritchie, & Wirth).

## ✨ Live Demo

```
chmod +x affirmation.sh
./affirmation.sh
```

## 🙏 Why This Works

Traditional "grace" prayers calm nerves with ritual. This tech adaptation:

1. **Builds anticipation** (15s prayer + animation)
2. **Visual payoff** (legends emerge)
3. **Memorable** (audience remembers the transition)

**Perfect for Tech demos, conferences, Twitch streams, YouTube tutorials.**
